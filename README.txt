Principais são o index e o about, secundários são os visíveis na tela inicial e terciários são os que são visíveis depois do about.

Nota: em aboutComprarProduto eu utilizei uma tag chamada <pre>, pelo o que eu entendi a identação dela é um pouco diferente. O texto dentro dela vai parecer esquisito dentro do código, mas não significa que está mal identado.

Nota0: em meus forms eu abreviei e excluí coisas inúteis, tais como classes desnecessárias; eu poli todos os forms, menos os de "aboutComprarProduto" pois a estrutura do código depende de classes diversificadas.

Aviso: no mobile o site fica bugado, foi projetado para ser usado em tela cheia. Em atualizações futuras eu vou arrumar alguns bugs(algum tempo após a entrega e resultado da nota), para o projeto ficar bonitinho.
